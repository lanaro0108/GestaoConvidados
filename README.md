# Sistema de Credenciamento de Evento

Projeto simples de uma página web para cadastro e gerenciamento de convidados em eventos.

## Funcionalidades

* Cadastro de convidados com:

  * Nome
  * E-mail
  * Telefone
  * Empresa
  * Tipo de ingresso

* Validação de campos obrigatórios
* Lista dinâmica de convidados (sem recarregar a página)
* Remoção individual de convidados
* Contador automático de participantes
* Modo escuro 🌙
* Limpeza do formulário ao pressionar **ESC**

---

## 🚀 Tecnologias Utilizadas

* HTML
* CSS
* JavaScript (DOM)

---

## 💻 Como usar

1. Clone o repositório:

```bash
git clone https://github.com/lanaro0108/GestaoConvidados.git
```

2. Abra o arquivo:

```bash
index.html
```

3. Use o sistema:

* Preencha o formulário
* Clique em **Confirmar**
* Veja o convidado aparecer na lista
